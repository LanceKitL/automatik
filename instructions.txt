-> GETTING STARTED <-

- before you test the program, it is required to run the xampp.
- make sure you matched the database name according to the "conn.py"
- install the dependencies indicated on the requirements.txt

to install all dependencies you can use this command:
"pip install -r requirements.txt"


to get the latest update on your current branch do this:
- git checkout main 
- git pull
- git checkout <branchName>
- git fetch origin
- git rebase origin/main

thats it! happy coding.
